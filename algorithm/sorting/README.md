# Sorting Algorithms

## Selection Sort

**Time complexity**: O(N^2)
    
    T(n) = T(n-1) + O(n)

**Space complexity**: O(1)

** Steps :**
    
    1. Find a largest number in A[:i + 1] and swap it to A[i]
    2. i = i - 1
    3. Recursively sort prefix A[:i], until i == 1
